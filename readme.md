# Emma Kostecki GitHub Pages Site

This repository hosts a personal portfolio and resume-style site for Emma, with
space to highlight projects, experience, and contact info.

## Live Site

Once GitHub Pages is enabled for this repo, the site will be available at:

`https://emmakostecki.github.io/`

## Project Structure

- `index.html` - main one-page portfolio layout
- `styles.css` - base styles and responsive layout

## Local Editing Workflow

1. Open the repo in Cursor.
2. Edit `index.html` content sections:
   - Hero intro
   - Featured projects
   - Experience highlights
   - Contact links
3. Adjust colors/layout in `styles.css`.
4. Preview in browser by opening `index.html` directly.

## Publish Changes

1. Commit changes to `main`.
2. Push to GitHub.
3. GitHub Pages automatically rebuilds and deploys.
